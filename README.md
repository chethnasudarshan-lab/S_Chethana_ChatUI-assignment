# Chat UI Project
**Student Name:** S Chethana
**Assignment:** Front-End Chat Interface

## About The Project
This is my submission for the Chat UI project. I made the front end of a chat application that acts like an AI bot. 

I built it using:
- HTML
- CSS
- JavaScript (with jQuery)
- Bootstrap 5 for some grid logic and standard styles
- FontAwesome for icons

There is no real backend server. The bot is just responding with random messages from an array in Javascript.

## Features
- **Responsive:** Looks okay on a computer and works on mobile (the sidebar hides like a toggle menu).
- **Chatting:** You can type messages and send them. 
- **Fake Bot:** It shows a "Bot is typing..." text, waits a second to feel real, and then replies.
- **Shortcuts:** The cards on the start screen can be clicked to fill the input box automatically.
- **Clear Chat:** The "New Chat" button clears the whole screen.

## How to Run it
1. Unzip this folder onto your desktop.
2. Just double-click on the `index.html` file to open it in your web browser (Chrome recommended).
3. Important: You must have an internet connection or else Bootstrap and the icons won't load from their CDNs!
